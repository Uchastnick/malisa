Умный дом

Позиционируется как open-source голосовой помощник для гиков. 

С возможностью расширения функционала благодаря сообществу (пока - русскоязычному),  путём написания отдельных action-процедур на языке python. 

Предоставляет возможности быстрой конфигурации шаблонов распознавания речи и реакций на них, составлять собственные плейлисты для музыкального проигрывателя, предоставляет возможность внятного чтения вслух текстов из файлов txt, fb2, pdf. 

А также предоставляя возможности пользоваться людям, далёким от программирования, после проведения минимально необходимых настроек.

В настоящий момент нацелен на русскоязычных пользователей систем Windows актуальных версий.
